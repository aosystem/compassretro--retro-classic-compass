# Retro Classic Compass 🧭

A reliable and high-precision **Digital Compass** with a classic aesthetic, built with **Flutter**. This app combines modern sensor technology with a timeless design, making it an essential tool for navigation, hiking, and travel.

## 🌟 Key Features
- **Accurate Orientation:** Real-time heading data using device hardware sensors.
- **Classic Retro UI:** A beautiful, easy-to-read vintage compass interface.
- **Global Support:** Fully localized in **50+ languages** for users around the world.
- **Privacy Focused:** Compass data is processed entirely on-device. No location tracking is shared.

## 💡 Best For...
- **Outdoor Exploration:** Hiking, camping, and trekking.
- **Travel & Navigation:** Quickly finding your way in new cities or environments.
- **Visual Design Fans:** Users who appreciate a mix of modern utility and classic style.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Integration with device magnetometer and accelerometer sensors for stable heading calculation.

*Note: This repository contains the core logic and source code. Assets (images, fonts, and localization files) are excluded to protect intellectual property.*
